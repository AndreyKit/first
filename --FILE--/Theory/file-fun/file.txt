<div class="floatright"><a href="/wiki/%D0%9F%D1%80%D0%BE%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%92%D0%B5%D0%BD%D0%B5%D1%80%D1%8B_%D0%BF%D0%BE_%D0%B4%D0%B8%D1%81%D0%BA%D1%83_%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B0" title="Прохождение Венеры по диску Солнца, случившееся в 2004 году."><img alt="Прохождение Венеры по диску Солнца, случившееся в 2004 году." src="//upload.wikimedia.org/wikipedia/commons/thumb/d/db/Venustransit_2004-06-08_07-49.jpg/160px-Venustransit_2004-06-08_07-49.jpg" decoding="async" width="160" height="164" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/db/Venustransit_2004-06-08_07-49.jpg/240px-Venustransit_2004-06-08_07-49.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/db/Venustransit_2004-06-08_07-49.jpg/320px-Venustransit_2004-06-08_07-49.jpg 2x" data-file-width="1375" data-file-height="1406" /></a></div>
<p><b><a href="/wiki/%D0%9F%D1%80%D0%BE%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%92%D0%B5%D0%BD%D0%B5%D1%80%D1%8B_%D0%BF%D0%BE_%D0%B4%D0%B8%D1%81%D0%BA%D1%83_%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B0" title="Прохождение Венеры по диску Солнца">Прохождение Венеры по диску Солнца</a></b>&#160;— разновидность <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_(%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%8F)" title="Прохождение (астрономия)">астрономического прохождения</a> (транзита),&#160;— имеет место тогда, когда <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0" title="Планета">планета</a> <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%80%D0%B0_(%D0%BF%D0%BB%D0%B0%D0%BD%D0%B5%D1%82%D0%B0)" class="mw-redirect" title="Венера (планета)">Венера</a> находится точно между <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B5" title="Солнце">Солнцем</a> и <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F" title="Земля">Землёй</a>, закрывая собой крошечную часть солнечного диска. При этом планета выглядит с Земли как маленькое чёрное пятнышко, перемещающееся по Солнцу. Прохождения схожи с <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D0%B5%D1%87%D0%BD%D0%BE%D0%B5_%D0%B7%D0%B0%D1%82%D0%BC%D0%B5%D0%BD%D0%B8%D0%B5" title="Солнечное затмение">солнечными затмениями</a>, когда наша звезда закрывается <a href="/wiki/%D0%9B%D1%83%D0%BD%D0%B0" title="Луна">Луной</a>, но хотя <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BC%D0%B5%D1%82%D1%80" title="Диаметр">диаметр</a> Венеры почти в 4 раза больше, чем у Луны, во время прохождения она выглядит примерно в 30 раз меньше Солнца, так как находится значительно дальше от Земли, чем Луна. Такой видимый размер Венеры делает её доступной для наблюдений даже невооружённым глазом (только с фильтрами от яркого солнечного света), в виде точки, на пределе разрешающей способности глаза. До наступления <a href="/wiki/%D0%9A%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%8D%D0%BF%D0%BE%D1%85%D0%B0" class="mw-redirect" title="Космическая эпоха">эпохи покорения космоса</a> наблюдения этого явления позволили <a href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%BE%D0%BC" title="Астроном">астрономам</a> вычислить расстояние от Земли до Солнца методом <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D0%BB%D0%BB%D0%B0%D0%BA%D1%81" title="Параллакс">параллакса</a>, кроме того, при наблюдении прохождения 1761 года <a href="/wiki/%D0%9B%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D1%81%D0%BE%D0%B2,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Ломоносов, Михаил Васильевич">М.&#160;В.&#160;Ломоносов</a> открыл <a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%B0_%D0%92%D0%B5%D0%BD%D0%B5%D1%80%D1%8B" title="Атмосфера Венеры">атмосферу Венеры</a>.
</p><p>Продолжительность прохождения обычно составляет несколько <a href="/wiki/%D0%A7%D0%B0%D1%81" title="Час">часов</a> (в <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 году</a> оно длилось 6 часов). В то же время, это одно из самых редких <i>предсказуемых</i> астрономических явлений. Каждые 243 <a href="/wiki/%D0%93%D0%BE%D0%B4" title="Год">года</a> повторяются 4 прохождения: два в декабре (с разницей в 8 лет), затем промежуток в 121,5 года, ещё два в июне (опять с разницей 8 лет) и промежуток в 105,5 года. Последние декабрьские прохождения произошли <a href="/wiki/9_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="9 декабря">9 декабря</a> <a href="/wiki/1874_%D0%B3%D0%BE%D0%B4" title="1874 год">1874 года</a> и <a href="/wiki/6_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="6 декабря">6 декабря</a> <a href="/wiki/1882_%D0%B3%D0%BE%D0%B4" title="1882 год">1882 года</a>, а июньские&#160;— <a href="/wiki/8_%D0%B8%D1%8E%D0%BD%D1%8F" title="8 июня">8 июня</a> <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 года</a> и <a href="/wiki/6_%D0%B8%D1%8E%D0%BD%D1%8F" title="6 июня">6 июня</a> <a href="/wiki/2012_%D0%B3%D0%BE%D0%B4" title="2012 год">2012 года</a>. Последующие прохождения произойдут в 2117 и 2125 годах, опять в декабре. Во время прохождения наблюдается «<a href="/wiki/%D0%AF%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%9B%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D1%81%D0%BE%D0%B2%D0%B0" title="Явление Ломоносова">явление Ломоносова</a>», а также «<a href="/wiki/%D0%AD%D1%84%D1%84%D0%B5%D0%BA%D1%82_%D1%87%D1%91%D1%80%D0%BD%D0%BE%D0%B9_%D0%BA%D0%B0%D0%BF%D0%BB%D0%B8" title="Эффект чёрной капли">эффект чёрной капли</a>».
</p>

<div id="main-tga" class="main-box main-box-responsive-image"><div class="main-box-subtitle">Хорошая статья</div><h2 class="main-header main-box-header"><span id=".D0.9C.D0.BE.D0.B4.D0.B5.D0.BB.D1.8C_.D0.BF.D0.B5.D1.80.D0.B5.D1.81.D0.B5.D0.BA.D0.B0.D1.8E.D1.89.D0.B8.D1.85.D1.81.D1.8F_.D0.BF.D0.BE.D0.BA.D0.BE.D0.BB.D0.B5.D0.BD.D0.B8.D0.B9"></span><span class="mw-headline" id="Модель_пересекающихся_поколений"><a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C_%D0%BF%D0%B5%D1%80%D0%B5%D1%81%D0%B5%D0%BA%D0%B0%D1%8E%D1%89%D0%B8%D1%85%D1%81%D1%8F_%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B9" title="Модель пересекающихся поколений">Модель пересекающихся поколений</a></span></h2>
<div class="floatright"><a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C_%D0%BF%D0%B5%D1%80%D0%B5%D1%81%D0%B5%D0%BA%D0%B0%D1%8E%D1%89%D0%B8%D1%85%D1%81%D1%8F_%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B9" title="производственная функция Кобба — Дугласа, логарифмическая функция полезности: достижение равновесия"><img alt="производственная функция Кобба — Дугласа, логарифмическая функция полезности: достижение равновесия" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Phase_diagram%28OG%2CK-D%2CLn%29.jpg/170px-Phase_diagram%28OG%2CK-D%2CLn%29.jpg" decoding="async" width="170" height="155" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Phase_diagram%28OG%2CK-D%2CLn%29.jpg/255px-Phase_diagram%28OG%2CK-D%2CLn%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Phase_diagram%28OG%2CK-D%2CLn%29.jpg/340px-Phase_diagram%28OG%2CK-D%2CLn%29.jpg 2x" data-file-width="544" data-file-height="495" /></a></div>
<p><b><a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C_%D0%BF%D0%B5%D1%80%D0%B5%D1%81%D0%B5%D0%BA%D0%B0%D1%8E%D1%89%D0%B8%D1%85%D1%81%D1%8F_%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B9" title="Модель пересекающихся поколений">Модель пересекающихся (перекрывающихся) поколений</a></b> (<i>модель Даймонда</i>, <i>модель Самуэльсона&#160;— Даймонда</i>, <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">overlapping generations model</span>)&#160;— <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C" title="Экономическая модель">модель</a> <a href="/wiki/%D0%AD%D0%BA%D0%B7%D0%BE%D0%B3%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Экзогенность">экзогенного</a> <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%BE%D1%81%D1%82" title="Экономический рост">экономического роста</a> в условиях <a href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%80%D1%88%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D0%BD%D0%BA%D1%83%D1%80%D0%B5%D0%BD%D1%86%D0%B8%D1%8F" title="Совершенная конкуренция">совершенной конкуренции</a>. Внесла вклад в понимание того, каким образом решения индивидов формируют норму <a href="/wiki/%D0%A1%D0%B1%D0%B5%D1%80%D0%B5%D0%B6%D0%B5%D0%BD%D0%B8%D1%8F" title="Сбережения">сбережений</a> в <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0" title="Экономика">экономике</a>. В модели отражено изменение потребительского поведение индивида по мере взросления. Вместе с тем, в модели отрицаются <a href="/wiki/%D0%90%D0%BB%D1%8C%D1%82%D1%80%D1%83%D0%B8%D0%B7%D0%BC" title="Альтруизм">альтруистические</a> связи между поколениями, и она не даёт удовлетворительного объяснения межстрановым различиям в уровне дохода на душу населения. Разработана <a href="/wiki/%D0%94%D0%B0%D0%B9%D0%BC%D0%BE%D0%BD%D0%B4,_%D0%9F%D0%B8%D1%82%D0%B5%D1%80" title="Даймонд, Питер">Питером Даймондом</a> с использованием идей <a href="/wiki/%D0%A1%D0%B0%D0%BC%D1%83%D1%8D%D0%BB%D1%8C%D1%81%D0%BE%D0%BD,_%D0%9F%D0%BE%D0%BB" title="Самуэльсон, Пол">Пола Самуэльсона</a> в 1965 году.
</p><p>Модификация модели с непрерывным временем, в которой жизнь индивида не делится на периоды молодости и старости, однако индивид может умереть в любой момент с некоторой вероятностью получила название «модель вечной молодости». Модель пересекающихся поколений более реалистично описывает общее экономическое равновесие и процесс его достижения, чем <a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C_%D0%A1%D0%BE%D0%BB%D0%BE%D1%83" title="Модель Солоу">модели Солоу</a> или <a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D1%8C_%D0%A0%D0%B0%D0%BC%D1%81%D0%B5%D1%8F_%E2%80%94_%D0%9A%D0%B0%D1%81%D1%81%D0%B0_%E2%80%94_%D0%9A%D1%83%D0%BF%D0%BC%D0%B0%D0%BD%D1%81%D0%B0" title="Модель Рамсея — Касса — Купманса">Рамсея — Касса — Купманса</a>.
</p>

<div class="floatright"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Elizaveta_Borissovna_Shakhovskaya_(1773-1796)_by_Peter_Edward_Stroely_(1768-1826).jpg" class="image"><img alt="Elizaveta Borissovna Shakhovskaya (1773-1796) by Peter Edward Stroely (1768-1826).jpg" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f0/Elizaveta_Borissovna_Shakhovskaya_%281773-1796%29_by_Peter_Edward_Stroely_%281768-1826%29.jpg/90px-Elizaveta_Borissovna_Shakhovskaya_%281773-1796%29_by_Peter_Edward_Stroely_%281768-1826%29.jpg" decoding="async" width="90" height="107" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f0/Elizaveta_Borissovna_Shakhovskaya_%281773-1796%29_by_Peter_Edward_Stroely_%281768-1826%29.jpg/135px-Elizaveta_Borissovna_Shakhovskaya_%281773-1796%29_by_Peter_Edward_Stroely_%281768-1826%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f0/Elizaveta_Borissovna_Shakhovskaya_%281773-1796%29_by_Peter_Edward_Stroely_%281768-1826%29.jpg/180px-Elizaveta_Borissovna_Shakhovskaya_%281773-1796%29_by_Peter_Edward_Stroely_%281768-1826%29.jpg 2x" data-file-width="1853" data-file-height="2200" /></a></div>
<ul><li>Брак дочери <span class="nowrap"><i>(на илл.)</i></span> с принцем обернулся для <b><a href="/wiki/%D0%A8%D0%B0%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F,_%D0%92%D0%B0%D1%80%D0%B2%D0%B0%D1%80%D0%B0_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%BD%D0%B0" title="Шаховская, Варвара Александровна">матери</a></b> инсультом и парализованной ногой.</li>
<li>Тяжёлые доспехи погубили <b><a href="/wiki/%D0%A0%D0%BE%D0%B4%D0%B6%D0%B5%D1%80_III_%D0%B4%D0%B5_%D0%9A%D0%BB%D0%B8%D1%84%D1%84%D0%BE%D1%80%D0%B4" title="Роджер III де Клиффорд">владельца</a></b> на переправе через <a href="/wiki/%D0%9C%D0%B5%D0%BD%D0%B0%D0%B9" title="Менай">уэльский пролив</a>.</li>
<li>В <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%BE%D0%B9_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Немой кинематограф">немом</a> фильме <b>«<a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BC%D0%B5%D0%BD_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC_%D0%94%D0%B5%D0%BC%D0%B8%D0%BB%D0%BB%D1%8F)" title="Кармен (фильм Демилля)">Кармен</a>»</b> заглавную роль исполняла <a href="/wiki/%D0%A4%D0%B0%D1%80%D1%80%D0%B0%D1%80,_%D0%94%D0%B6%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%B4%D0%B8%D0%BD%D0%B0" title="Фаррар, Джеральдина">оперная примадонна</a>.</li>
<li><b><a href="/wiki/%D0%96%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D1%80%D1%82%D1%80%D0%B5%D1%82_(%D0%BA%D0%B0%D1%80%D1%82%D0%B8%D0%BD%D0%B0_%D0%B2%D0%B0%D0%BD_%D0%92%D0%BB%D0%B8%D1%82%D0%B0)" title="Женский портрет (картина ван Влита)">Картину</a></b> работы <b><a href="/wiki/%D0%92%D0%BB%D0%B8%D1%82,_%D0%A5%D0%B5%D0%BD%D0%B4%D1%80%D0%B8%D0%BA_%D0%9A%D0%BE%D1%80%D0%BD%D0%B5%D0%BB%D0%B8%D1%81_%D0%B2%D0%B0%D0%BD" title="Влит, Хендрик Корнелис ван">художника XVII века</a></b> в России чуть не сожгли, как «наследие старого режима».</li>
<li>Американский судья заявил, что танец <b><a href="/wiki/%D0%A1%D0%B0%D0%BB%D0%BB%D0%B8_%D0%A0%D1%8D%D0%BD%D0%B4" title="Салли Рэнд">стриптизёрши</a></b> кажется непристойным только обладателям извращённой морали.</li>
<li>У <a href="/wiki/%D0%9E%D0%B1%D1%8B%D0%BA%D0%BD%D0%BE%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B1%D0%B5%D0%B3%D0%B5%D0%BC%D0%BE%D1%82" title="Обыкновенный бегемот">бегемотов</a> есть <b><a href="/wiki/%D0%9B%D0%B0%D0%B1%D0%B5%D0%BE" title="Лабео">своя</a></b> <a href="/wiki/%D0%A0%D1%8B%D0%B1%D1%8B-%D1%87%D0%B8%D1%81%D1%82%D0%B8%D0%BB%D1%8C%D1%89%D0%B8%D0%BA%D0%B8" title="Рыбы-чистильщики">рыба-чистильщик</a>.</li></ul>
